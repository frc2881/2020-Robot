# 2020RobotDrive
Java source code for the FRC competition robot

PS4 Dualshock Button Mapping:
* 0 = Pink Square
* 1 = Blue X
* 2 = Red Circle
* 3 = Green Triangle
* 4 = Left Bumper
* 5 = Right Bumper
* 6 = Left Trigger
* 7 = Right Trigger
* 8 = Share
* 9 = Options
* 10 = Left Joystick Press (L3)
* 11 = Right Joystick Press (R3)
* 12 = Playstation Button (Home)
* 13 = Touch Pad Button

PS4 Dualshock Raw Axis Mapping:
* 0 = Left X
* 1 = Left Y
* 2 = Right X
* 3 = Left Trigger
* 4 = Right Trigger
* 5 = Right Y
